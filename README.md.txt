# Student JSON Manager

A beginner Python project that reads, processes, and updates student data using a JSON file.

## Features

- Show all students
- Count all students
- Calculate average age
- Find the oldest student
- Find the youngest student
- Add a new student
- Save updated student data into a JSON file

## Skills Used

- Python
- JSON
- File handling
- Lists
- Dictionaries
- Loops
- Conditions
- `while True` menu
- `json.load()`
- `json.dump()`

## How It Works

The program reads student data from `students.json`, then shows a menu where the user can choose different actions.

The user can view students, count them, calculate average age, find the oldest or youngest student, and add a new student.

When a new student is added, the program saves the updated data back into `students.json`.

## Project Goal

This project was built to practice working with structured data using Python and JSON files. It helps improve beginner Python logic, data processing, and file handling skills.
